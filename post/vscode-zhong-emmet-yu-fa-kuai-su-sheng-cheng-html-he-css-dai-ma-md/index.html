VsCode 中 Emmet 语法快速生成HTML 和 CSS 代码.md

· 2020-09-30 · # VsCode # CSS # HTML # 前端

[TOC]

敲完简写后，按 Tab 自动完成

HTML

  • div
<div></div>
  • div*3
<div></div>
<div></div>
<div></div>
  • {} 定义标签内的内容，div{Hello}
<div>Hello</div>
  • div{Hello}*3
<div>Hello</div>
<div>Hello</div>
<div>Hello</div>
  • 父子关系标签，使用 >，如 ul>li：
<ul>
    <li></li>
</ul>

div>p：

<div>
    <p></p>
</div>
  • 兄弟关系标签，使用 +，如 div+p：
<div></div>
<p></p>
  • () 指定优先级，如 ul>(li+bq)：
<ul>
    <li></li>
    <blockquote></blockquote>
</ul>

(ul>li)+bq：

<ul>
    <li></li>
</ul>
<blockquote></blockquote>
  • 带有类选择器，如 .gray，默认生成 class 为 gray 的 div：
<div class="gray"></div>

如果要生成其它标签 class 为 gray，则 p.gray：

<p class="gray"></p>

一次生成多个 class 选择器，如 .box1.box2.box3：

<div class="box1 box2 box3"></div>

p.box1.box2.box3：

<p class="box1 box2 box3"></p>
  • 带有 ID 选择器，如 #gray，默认生成 id 为 gray 的 div：
<div id="gray"></div>

如果要生成其它标签 id 为 gray，则 p#gray：

<p id="gray"></p>

既有类选择器，又有 id 选择器，如 p.box1.box2#gray：

<p class="box1 box2" id="gray"></p>
  • [] 表示属性，如 ul>li>a[href="#"]：
<ul>
    <li><a href="#"></a></li>
</ul>
  • $ 表示自增，如 div{$}*3：
<div>1</div>
<div>2</div>
<div>3</div>

div.gray$*3：

<div class="gray1"></div>
<div class="gray2"></div>
<div class="gray3"></div>

p.gray${$}*3：

<p class="gray1">1</p>
<p class="gray2">2</p>
<p class="gray3">3</p>

ul>li.gray${Hello$}*3：

<ul>
    <li class="gray1">Hello1</li>
    <li class="gray2">Hello2</li>
    <li class="gray3">Hello3</li>
</ul>

CSS

CSS 中一般取每个单词首字母，然后 Tab 即可自动补全/生成：

  • w200：
width: 200px;
  • h200：
height: 200px;
  • lh26px：
line-height: 26px;
  • ti2em：
text-indent: 2em;
  • tdn：
text-decoration: none;
  • tac：
text-align: center;
